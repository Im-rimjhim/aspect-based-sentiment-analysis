# Aspect-Based Sentiment Analysis for E-Commerce Reviews

This project extracts product aspects like quality and delivery from e-commerce reviews and analyzes the sentiment for each aspect.

## Files
- `Aspect_Based_Sentiment_Analysis.ipynb`: Notebook with the full code.
- `sample_reviews.csv`: Sample review data.

## How to run
Open the notebook in Google Colab or Jupyter and run all the cells.
